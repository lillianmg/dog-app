# Dog Walking App

[Live Site](https://dogapp.lilliangil.com/) • [GitHub Repo](https://github.com/lillianmg/dog-app)

A React app that checks current weather and tells you if it is safe to walk your dog.

## Features
- Search by city
- See current temperature in Fahrenheit
- Get a dog-walk safety message based on temperature and dog size

## Tech Stack
React, JavaScript, HTML, CSS, Axios, GitHub Pages

## Deployment
Hosted on GitHub Pages with a custom domain: `dogapp.lilliangil.com`

## Run Locally
```bash
npm start
npm test
npm run build
npm run deploy
```
